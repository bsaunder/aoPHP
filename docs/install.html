INSTALL AOPHP
To Install aophp Follow the Steps Below

  1. Download aophp
  2. Extract Archive
    tar -xf aophp_v3_1.tar
  3. cd into the created directory
    cd     vx_x_x/
  4. cd into the upload directory
    cd upload/
  5. Modify the aophpFilePath and outputFileDir variables in the aophpexec.aophp file
    1. The aophpFilePath variable should be the full path to the aophp binary
    2. The outputFileDir variable should be the full path to the directory where the weaved files will be written. This directory should be writable by the aophp binary and access able by the aophpexec script. You may need to chmod this directory using the chmod command.
  6. Upload the aophpexec.aophp and .htaccess files to every directory where you want to use aoPHP. For instance if your application is in the WEBROOT/myapp directory, upload them there.
  7. Upload the aophp binary file to the directory that you set in aophpFilePath variable
    1. If you need to compile the binary, follow the instructions for compiling a binary below

COMPILE AOPHP
Follow the steps below to compile aoPHP from the source code.

  1. Download the Boost Libraries, Boost Jam, and Xpressive from the aophp website or get the latest version from www.boost.org
  2. Install Boost Jam
    1. Decompress and Extract the archive
      gunzip boost-jam-3.1.12.tgz
      tar -xf boost-jam-3.1.12.tar
    2. cd into the created directory
    3. run the build script
      sh ./build.sh
    4. Go into the newly created directory containing the bjam executable (bin.linuxx86 on my system) and copy the bjam executable to /bin
      cd bin.linuxx86
      cp bjam /bin
  3. Install Boost Libraries
    1. Decompress and Extract the archive
      gunzip boost_1_33_1.tar.gz
      tar -xf boost_1_33_1.tar
    2. cd into the created directory
      cd boost_1_33_1
    3. Configure the Libraries
      ./configure
    4. Run the installation script
      bjam "-sTOOLS=gcc" install
    5. Go get some coffee, its going to take a minute
  4. Install Xpressive
    1. Extract the archive to your include directory
      unzip xpressive.zip -d /usr/include
  5. Download aophp Source Code
  6. Extract Archive
    tar -xf aophp_v3_1.tar
  7. cd into the created directory
    cd     vx_x_x/
  8. cd into the src directory
    cd src/
  9. Compile aophp
    make install

Note: You can follow the official instructions for compiling and installing Boost/Boost Jam by going to this link: http://www.boost.org/more/getting_started.html
