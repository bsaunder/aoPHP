Syntax and Valid Types

Pointcut Types:

  • exec
  • execr
  • set
  • get

Advice Types:

  • before
  • around
  • after

Sample Syntax

aspect NAME {

    pointcut PCNAME = execr(file.func($p1,$p2));
    pointcut PCNAME2 = set(file.x);

    advice before($ip1,$ip2): PCNAME {%
        ... CODE ...
    %}

    advice around(): PCNAME2 | set(file.y) {%
        ... CODE ...
    %}

    adivce after(): exec(file.func2())&&arg1(5)&&arg2("Word") {%
        ... CODE ...
    %}
}
